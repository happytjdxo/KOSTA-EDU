생성자에 대해서

"객체가 생성 될 때 마다 호출(구동) 된다."
	1. 생성자는 크게 기본생성자 / 명시적 생성자로 나뉜다.
	2. 기본생성자는 개발자가 안넣더라도 컴파일러가 대신 넣고 돌린다.
	3. 기본생성자는 인자값이 없고, 구현부에서 어떤 일도 하지 않는 생성자이다.
	4. 명시적 생성자가 하는일은 필드 초기화
	5. Test 클래스에서 생성자의 값이 전달될 때 인자값의 순서, 갯수, 타입을
	   지켜서 Passing 해줘야 한다,
	6. 생성자는 클래스 이름과 동일하고 클래스의 접근지정자를 따른다.
	7. 하나의 클래스에는 생성자가 반드시 하나 이상이어야 한다.
